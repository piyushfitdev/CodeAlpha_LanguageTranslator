# Language Translation Tool

A Python-based Language Translation Tool built using Tkinter and Google Translator API.

## Features
- Translate text between multiple languages
- Simple GUI interface
- Copy translated text
- Text-to-Speech functionality

## Technologies Used
- Python
- Tkinter
- deep-translator
- pyttsx3

## Installation

```bash
pip install -r requirements.txt
```

## Run the Project

```bash
python main.py
```
